## כללי עיצוב קבועים לפלט HTML

> כללים מחייבים שיש ליישם אוטומטית בכל פלט HTML, בכל סשן, גם אם שאר מדריך הסגנון עדיין לא מולא.

- **כל הכותרות (h1–h6) צבועות בירוק.** בכל קובץ HTML שמופק, יש לוודא שכל רמות הכותרות
  מופיעות בצבע ירוק. מומלץ להגדיר זאת ב-CSS באמצעות כלל אחד מרכז, למשל:
  ```css
  h1, h2, h3, h4, h5, h6 { color: #2e7d32; }
  ```
  אם קיים בקובץ כלל ספציפי שמגדיר צבע אחר לכותרת כלשהי (למשל `h3 { color: ... }`),
  יש לעדכן גם אותו לירוק כדי שלא ידרוס את הכלל הכללי.
